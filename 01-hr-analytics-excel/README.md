# 🧩 Project 1: HR Analytics Dashboard (Excel)

**File:** `2nd_Assigment_-_HRDataSet.xlsx`

An HR analytics workbook built with pivot tables to explore workforce data for **998 employees** across:

- **Branches:** Cairo, Giza, Alexandria, Sharqia, Ismailia
- **Departments:** Manufacturing, Maintenance, Administration, Marketing, Training, HR, Operations, Quality Assurance, Compliance, Logistics, SS
- **Employment Type:** Full Time / Part Time

## Analyses included
- Employees by Branch
- Age Demographics (grouped ranges)
- Department Distribution
- Top 10 Salaries (with bonus breakdown)
- Employment Type Analysis (salary & headcount by Full/Part Time)
- Gender Diversity & Average Pay

## Workbook structure
| Sheet | Content |
|---|---|
| `Sheet1` | Quick summary view |
| `Pivot Tables Analysis` | The five pivot table analyses listed above |
| `HR_DB4` | Raw employee-level dataset (16 columns, 998 rows) |

**Skills demonstrated:** Excel pivot tables, pivot charts, data cleaning, HR/compensation analysis.
